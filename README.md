 # Speak-fluently 流利说外语
 
 ## 什么是 Speak-fluently?
 这是一个[TamperMonkey](https://www.tampermonkey.net/)的脚本。主要目的是帮助大家流畅的练习`说`外语。
 
 ## 那到底怎么帮助我们练习外语呢？
 Youtube是一个很好的外语内容来源， 这个脚本通过在Youtube上添加复读，录音等功能，帮助大家练习口语。
 
 ### 听
 YouTube上的视屏内容都可以作为练习的材料。
 
 ### 说 && 对比
 听完一段话之后， 你可以自己录音，然后比对录音和原音之间的区别。
